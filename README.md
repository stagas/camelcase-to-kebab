## Install

```sh
$ npm i camelcase-to-kebab
```

## Example

```ts
import { camelCaseToKebab } from 'camelcase-to-kebab'

console.log(camelCaseToKebab('helloKebabWorld'))
// => hello-kebab-world
```

## Bench
